# ML Stock Pricing Signals Project

This project focuses on predicting stock prices using machine learning techniques.

## Project Structure:
- data/: Contains datasets
- notebooks/: Contains Jupyter notebooks
- src/: Python scripts for processing and modeling
- configs/: Configuration files

## How to Run
1. Install dependencies using `pip install -r requirements.txt`
2. Run `streamlit run app.py` to start the project.
